---

Các thành phần chính:

1. **index.html**
   - Tải MediaPipe từ CDN
   - Layout: video + canvas overlay + controls
   - Nút: Upload, Prev/Next Frame, Undo/Redo, Zoom, Autosave, Extract, Save

2. **app.js** (~500 dòng)
   - State quản lý: frames, fps, view (zoom/pan), history (undo/redo)
   - Upload video/JSON handler
   - Canvas drawing: vẽ bones + keypoints
   - Mouse interactions: drag/drop, hover, pan
   - Undo/Redo logic
   - Autosave (localStorage)
   - Keyboard shortcuts (Ctrl+Z/Y)

3. **mediapipe-extractor.js** (~150 dòng)
   - Tải MediaPipe Hands từ CDN
   - Frame-by-frame video processing
   - Extract 21 hand landmarks per frame
   - Return array: [ frame0: [21 points], frame1: [21 points], ... ]

4. **style.css**
   - Minimal, responsive design
   - Canvas overlay positioning
   - Button styles + disabled states
   - Tooltip styles

5. **sample-keypoints.json**
   - 2 frames test data × 21 points
   - Format: [x: 0..1, y: 0..1, z: 0]
   - Dùng để test UI mà không cần video/extraction

---

Yêu cầu:

- Browser: Chrome/Edge/Firefox (hỗ trợ Canvas, MediaPipe)
- Video: MP4 format (H.264 codec)
- JSON: Array hoặc {frames: [...]}
- Internet: Để tải MediaPipe từ CDN (lần đầu)

---

Cấu hình:

- FPS: Mặc định 25, thay đổi ở ô "FPS" trên giao diện
- Point radius: 9px (có thể chỉnh trong app.js: state.pointRadius)
- Max undo per frame: 50 (có thể chỉnh trong app.js: history)
- Autosave key: "vsl_editor_autosave" (localStorage)
- Zoom range: 0.2x - 4x

Developer notes:

- Architecture: Vanilla JS IIFE (no build tool needed)
- State management: Single state object + history per frame
- Canvas: 2D context + view transform (scale, offset)
- View transform: ctx.setTransform(scale, 0, 0, scale, offsetX, offsetY)
- Hit testing: Distance-based (radius = 1.4x pointRadius)
- Drag normalization: Store [0..1], render at display size
- MediaPipe async: Uses Promise + frame seeking (video.currentTime = t)

---

Performance notes:

- Extraction: ~1-2 sec per frame (depends on GPU/CPU)
- Rendering: 60 FPS canvas draw loop
- Canvas size: Matches video resolution (usually 1280x720 or 640x480)
- Memory: Frames stored in state.frames; set to null to free
- Browser tab: Stay focused during extraction (MediaPipe performance affected)

---

Troubleshooting:

1. Check console (F12) for errors
2. Verify video format (MP4 H.264)
3. Check MediaPipe CDN reachable (network tab)
4. Ensure localStorage enabled in browser
5. Try with sample-keypoints.json first
6. Test with short video (< 10 sec) for extraction
